Troubleshooting Guide

Connection Issues
  • Check network stability before starting
  • Use wired connection if possible
  • Hard refresh with Ctrl+Shift+R
  • Disable VPN or proxy temporarily
  • Restart your router if unstable
Browser Issues
  • Clear cache and cookies first
  • Enable JavaScript (required)
  • Update to latest browser version
  • Try incognito / private mode
  • Disable all extensions temporarily
Fullscreen Problems
  • Allow fullscreen in browser prompt
  • Silence system notifications
  • Disable multiple monitors if possible
  • Don't use Alt+Tab during exam
  • Check browser fullscreen permissions
Formula Not Rendering
  • Wait for MathJax to fully load
  • Refresh page if symbols show
  • Use Chrome for best rendering
  • Disable math-related extensions
  • Report if issue persists
